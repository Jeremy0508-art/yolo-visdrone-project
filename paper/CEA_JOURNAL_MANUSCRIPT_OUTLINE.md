摘要按“目的-方法-结果-结论”组织：

- 方法：说明在 YOLO11n 中引入 P2 高分辨率检测分支、CoordAttention 和 960 输入分辨率，并补充小目标友好增强消融。
- 结果：只写已有真实结果；服务器公平对照完成后再补充对照结论。
- 结论：强调高分辨率输入是主要增益来源，P2 与 CoordAttention 提供结构补充，方法在精度和速度之间取得折中。

## 1 引言

### 1.1 背景

写作要点：

- 无人机在交通监控、应急巡检、城市管理、低空遥感中的应用。
- 航拍图像与地面图像相比有俯视视角、尺度变化、目标密集和遮挡问题。
- VisDrone2019-DET 是无人机目标检测常用数据集，目标类别包括行人、车辆和非机动车。

### 1.2 问题

写作要点：

- 轻量 YOLO 模型速度快，但深层特征图空间分辨率低，小目标信息容易丢失。
- 单纯增大模型容量会提高计算成本，不一定适合边缘部署。
- 高分辨率输入、浅层检测头和位置敏感注意力可能改善小目标检测，但需要用公平实验区分贡献来源。

### 1.3 本文贡献

建议贡献点：

1. 构建 HRPCA-YOLO11n，在 YOLO11n 中融合 P2 高分辨率检测分支与 CoordAttention，以增强浅层空间细节和位置敏感表达。
2. 建立同协议实验矩阵，比较 YOLO11n、YOLOv8n、YOLO11s 及分辨率公平对照，分析结构、分辨率和模型容量的影响。
3. 基于 VisDrone 标注和预测结果开展目标尺度分布与尺度分组匹配分析，验证模型收益与小目标场景的关系。
4. 从复杂度、速度、类别级表现和失败案例角度讨论方法适用范围。

## 3 方法

### 3.1 总体结构

写作要点：

- 输入图像经 YOLO11n backbone 提取特征。
- Neck 进行多尺度融合。
- 检测输出由 P3/P4/P5 扩展为 P2/P3/P4/P5。
- CoordAttention 位于特征融合分支中，增强位置敏感表达。
- 960 输入分辨率作为高分辨率策略。

需要图件：

- 方法结构图，建议以 P2/P3/P4/P5 四尺度输出为核心视觉信息。

### 3.2 P2 高分辨率检测分支

### 3.3 CoordAttention 模块
