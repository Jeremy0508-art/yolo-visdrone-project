# 方法章节草稿

本文面向无人机航拍场景中的小目标检测问题，在 YOLO11n 基线模型基础上构建轻量化改进模型。整体思路是：通过增加 P2 高分辨率检测分支增强小尺度目标感知能力，通过 CoordAttention 引入方向位置信息以改善复杂背景下的特征表达，并通过更高输入分辨率提升小目标在特征图中的有效像素占比。

## 基线模型

本文采用 Ultralytics YOLO11n 作为基线检测器。YOLO11n 具有较小参数量和较快推理速度，适合无人机巡检、交通监控等对部署效率有要求的场景。基线模型直接在 VisDrone2019-DET 数据集上训练，用于评估后续结构和训练策略改动的实际收益。

## P2 高分辨率检测分支

VisDrone 图像中存在大量远距离行人、车辆和非机动车目标。这类目标在原始图像中尺寸较小，经过多次下采样后容易在深层特征图中丢失细节。为缓解该问题，本文在 YOLO11n 检测头中引入 P2 高分辨率检测分支，使检测头同时利用更浅层、更高空间分辨率的特征。

项目中的 P2 模型配置文件为 `configs/models/yolo11n_p2.yaml`。该结构在原有多尺度特征融合基础上继续上采样，并与浅层特征进行拼接，形成额外的小目标检测尺度。最终 Detect 层使用四个尺度特征进行预测，对应配置中的：

```text
[[19, 22, 25, 28], 1, Detect, [nc]]
```

相比原始 YOLO11n，P2 分支会增加一定参数量和计算量，但能够保留更多小目标定位所需的细粒度纹理和边界信息。

## CoordAttention 注意力增强

在 P2 模型基础上，本文进一步引入 CoordAttention 模块。与普通通道注意力相比，CoordAttention 在建模通道关系的同时保留方向相关的位置信息，有助于模型在复杂背景和密集目标场景下区分目标区域与背景区域。

项目中的注意力模型配置文件为 `configs/models/yolo11n_p2_coordatt.yaml`。该配置在两个特征融合分支后加入 CoordAttention：

```text
[-1, 1, CoordAttention, [128, 32]]
[-1, 1, CoordAttention, [256, 32]]
```

加入注意力后的 Detect 层对应四尺度输出：

```text
[[19, 22, 26, 30], 1, Detect, [nc]]
```

该设计保持了轻量化目标，只在局部融合分支增加少量参数，避免显著降低推理效率。

## 高分辨率输入策略

由于 VisDrone 中小目标占比较高，单纯依赖检测头结构仍可能受到输入图像有效分辨率限制。因此，本文在 YOLO11n-P2-CoordAttention 基础上进一步测试 960 输入分辨率。该实验配置文件为 `configs/train/yolo11n_p2_coordatt_960.yaml`，主要设置为：

```text
imgsz: 960
batch: 4
epochs: 100
close_mosaic: 10
```

更高输入分辨率能够让小目标在网络输入和中间特征图上占据更多像素，从而提升定位和分类稳定性。该策略会增加显存和推理耗时，因此本文在实验部分同时报告精度、模型复杂度和速度结果，用于分析精度与效率之间的折中。

## 小目标友好数据增强

本文还设计了一个小目标友好数据增强消融实验，用于检验训练策略对小目标检测的影响。该实验配置文件为 `configs/train/yolo11n_p2_coordatt_smallobj_aug.yaml`，主要参数为：

```text
close_mosaic: 20
scale: 0.35
copy_paste: 0.1
erasing: 0.0
```

其中，`close_mosaic: 20` 表示在最后 20 个 epoch 关闭 mosaic，使模型更早回到真实图像分布；`scale: 0.35` 用于减小随机缩放范围，降低小目标被过度缩小后难以识别的风险；`copy_paste: 0.1` 提供轻量目标复制增强；`erasing: 0.0` 避免随机擦除破坏小目标可见区域。

需要注意的是，该增强策略是本文的训练策略消融项，而不是最终最优模型的必要组成部分。实验结果表明，它相较原始 YOLO11n 有提升，但不如 P2 结构和 960 输入分辨率带来的增益明显。

## 最终模型选择

综合验证集精度和速度测试结果，本文将 YOLO11n-P2-CoordAttention-960 作为当前已完成实验中的最佳模型。该模型保留 YOLO11n 的轻量化基础，通过 P2 分支增强小目标特征表达，通过 CoordAttention 改善空间与通道特征建模，并通过 960 输入分辨率进一步提升小目标检测精度。
