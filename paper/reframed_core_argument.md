# 重构后的论文核心论点

状态：用于中文中刊稿和英文扩展稿的共同主线。

## 一句话主线

本文研究轻量 YOLO 在无人机航拍小目标检测中采用高分辨率输入与浅层 P2 预测分支的收益、代价和适用边界，而不是简单宣称某个改进模块全面优于所有基线。

## 核心论点

轻量 YOLO 模型在无人机航拍小目标场景中面临两类矛盾：

1. 小目标需要更高空间分辨率和更浅层的细节特征；
2. 轻量部署又要求参数量、计算量和延迟不能大幅增加。

因此，高分辨率输入和 P2 浅层预测分支不是孤立的“结构改动”，而是一种围绕小目标细节保留的轻量化折中机制。本文的重点是评估这种机制在精度、尺度诊断、复杂度和跨数据集表现上的真实边界。

## 可以成立的贡献

以下贡献不依赖尚未完成的 UAVDT 最终结果：

1. **机制研究贡献**  
   系统分析输入分辨率、P2 浅层预测、注意力校准和轻量化复杂度之间的关系，明确高分辨率预测在无人机小目标检测中的收益来源和代价。

2. **证据链贡献**  
   建立从训练日志、验证结果、速度测试、模型复杂度、尺度分组诊断到论文表格的可追溯证据链，避免只报告单一 mAP。

3. **诊断指标贡献**  
   在常规 mAP50/mAP50-95 之外，加入小/中/大尺度 recall、precision 和本地 scale-bin AP 诊断，用于分析高分辨率预测对不同目标尺度的影响。

4. **边界讨论贡献**  
   明确指出轻量模型不应被包装成超过大模型的绝对最优方案，并将 YOLO11s 与 UAVDT 作为容量边界和数据集边界验证。

## 不能提前成立的贡献

以下表述必须等 UAVDT 或更多实验完成后再决定：

- 本方法具有稳定跨数据集泛化能力；
- P2 在所有无人机数据集上均优于 YOLO11n baseline；
- TOFC 是最终小目标增强主模块；
- 方法达到或接近 SOTA；
- 方法优于更大容量模型。

## 摘要改写原则

摘要不能写成“本文改了哪些参数”。应按以下逻辑：

1. 无人机航拍小目标检测中，轻量模型容易丢失浅层细节；
2. 高分辨率输入和 P2 预测可能缓解该问题，但会增加复杂度且跨数据集稳定性不确定；
3. 本文构建统一实验框架，分析高分辨率预测、注意力校准和复杂度之间的关系；
4. 基于真实日志报告 VisDrone 结果，并用 UAVDT 验证边界；
5. 结论强调收益、代价和适用条件。

## 结论改写原则

结论必须避免“全面提升”。推荐结构：

1. 在 VisDrone 上，高分辨率输入带来主要收益；
2. P2 在分辨率对齐后进一步改善部分小目标诊断指标；
3. 注意力和 TOFC 的效果具有指标依赖性，不能简单视为稳定增益；
4. 大模型仍具有明显绝对精度优势；
5. UAVDT 决定最终能否写成跨数据集方法贡献，否则应写成边界分析。
