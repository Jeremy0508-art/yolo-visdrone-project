## Build Command

Compile the current advisor-review PDF from the project root:

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\build_paper_pdf.ps1 -TexFile paper\ieee_trans\main_draft.tex -OutDir paper\ieee_trans
```

Output:

```text
paper/ieee_trans/main_draft.pdf
```

The project uses a repo-local Tectonic binary under `.tools/tectonic/`. The local files `IEEEtran.cls`, `IEEEtran.bst`, `cite.sty`, `TS1cmr.fd`, and `cmsy7.pfb` are kept in `paper/ieee_trans/` so the draft can compile even when Tectonic cannot fetch IEEE template or font resources from the network.

## UAVDT Result Gate

UAVDT rows are generated by a separate script:


This script should be used only after complete runs are synced. The current manuscript must not use partial server metrics. See `post_uavdt_rewrite_checklist.md` and `uavdt_result_integration.md`.

`paper/tables/ieee_uavdt_results_for_paper.csv`. This evidence is a boundary
result for static P2, not a ScaleGate result.
