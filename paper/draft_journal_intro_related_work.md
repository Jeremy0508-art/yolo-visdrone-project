## 1 引言

随着低空智能感知和无人机平台的发展，无人机航拍图像已广泛应用于城市交通监控、应急救援、公共安全巡检、道路拥堵分析和低空遥感等场景。与地面视角图像相比，无人机图像通常具有拍摄高度变化大、俯视角明显、目标尺度跨度大、背景纹理复杂和目标分布密集等特点。尤其在城市道路和开放场景中，行人、车辆、非机动车等目标往往只占据较少像素，且目标之间存在遮挡、截断和类别外观相似等现象，这给检测模型的特征表达、定位精度和类别判别能力带来较大挑战。

VisDrone2019-DET 是无人机目标检测领域常用的公开数据集之一，包含多种无人机拍摄高度、视角和场景，目标类别覆盖 pedestrian、people、car、van、bus、motor、bicycle、tricycle、awning-tricycle 和 truck 等十类目标。该数据集具有小目标占比高、密集目标多和类别间视觉差异小等特点，能够较好反映真实航拍检测任务中的困难。本文基于 YOLO 格式标注统计发现，VisDrone 验证集中 small 目标占比达到 68.59%，说明小目标检测能力是影响该任务整体性能的重要因素。

YOLO 系列检测器因端到端预测、推理速度快和工程部署便利等特点，在实时目标检测任务中得到广泛应用。Ultralytics YOLO11 延续了 YOLO 系列的实时检测优势，并提供不同规模的模型。YOLO11n 作为 nano 版本具有参数量小、推理成本低的特点，适合作为无人机边缘检测场景的轻量化基线。然而，轻量模型通常受到特征容量和空间分辨率限制。经过多次下采样后，小目标在深层特征图中的有效响应区域进一步缩小，浅层细节和边界信息容易丢失，从而导致远距离行人、非机动车和小型车辆等目标漏检。

针对无人机航拍小目标检测问题，现有研究通常从高分辨率输入、多尺度特征融合、浅层检测头、注意力机制和数据增强等方向进行改进。高分辨率输入可以直接增加小目标在输入图像中的有效像素占比；浅层检测头能够利用更高空间分辨率的特征进行预测；注意力机制则有助于在复杂背景中增强目标区域响应。但这些方法也会带来计算量增加、推理速度下降或模块收益有限等问题。因此，对于轻量化无人机检测模型而言，关键不只是追求单一精度提升，而是需要系统分析结构、输入分辨率和模型容量对精度、速度与复杂度的影响。

本文以 YOLO11n 为基线，围绕 VisDrone 无人机航拍小目标检测构建轻量化改进方法。首先，在检测头中引入 P2 高分辨率检测分支，使模型由 P3/P4/P5 三尺度预测扩展为 P2/P3/P4/P5 四尺度预测，从而增强浅层高分辨率特征对小目标定位的支持。其次，在特征融合分支中加入 CoordAttention 模块，利用方向位置敏感信息补充特征表达。最后，在 P2 与 CoordAttention 结构基础上评估 960 输入分辨率和小目标友好增强策略，并通过主线消融、外部基线、尺度分组分析、速度复杂度和可视化结果进行系统验证。

本文主要工作如下：

1. 面向 VisDrone 无人机航拍小目标检测任务，构建融合 P2 高分辨率检测分支与 CoordAttention 的轻量化 YOLO11n 改进模型，增强浅层空间细节和位置敏感特征表达。
2. 建立可复现的实验与证据链，系统比较 YOLO11n、YOLO11n-P2、YOLO11n-P2-CoordAttention、YOLO11n-P2-CoordAttention-960 和小目标友好增强配置，所有指标均来自真实训练日志、验证结果文件或脚本生成表格。
3. 补充 YOLOv8n、YOLO11s 及服务器公平分辨率对照实验，用于分析不同 YOLO 版本、模型容量和输入分辨率下的精度-速度关系。未完成的服务器实验不会进入论文主表。
4. 基于 VisDrone 标注和预测结果开展目标尺度分布与尺度分组匹配分析，说明小目标占比和不同尺度目标召回变化，为方法设计提供数据支撑。
5. 从类别级指标、模型复杂度、推理速度、精度-速度-参数量折中图和失败案例归因等角度分析模型优势与局限。

## 2 相关工作

### 2.1 无人机航拍目标检测

无人机航拍目标检测需要在高空俯视图像中定位和识别多类目标。与常规自然图像检测相比，无人机图像中的目标尺度通常更小，背景更复杂，目标分布更密集，并且常伴随遮挡和截断现象。VisDrone 等公开数据集推动了无人机目标检测算法的研究，也暴露出小目标漏检、密集目标分离困难和类别间混淆等问题。针对这些问题，研究者通常从特征融合、检测头设计、数据增强、注意力机制和后处理优化等方面改进检测模型。

在无人机检测场景中，轻量化模型具有实际部署价值。一方面，无人机或边缘设备通常算力和功耗受限，需要模型具有较小参数量和较快推理速度；另一方面，过度压缩模型又可能降低小目标检测能力。因此，无人机目标检测算法需要在检测精度、模型复杂度和推理效率之间取得平衡。本文选择 YOLO11n 作为基线模型，正是为了在轻量化框架下分析小目标增强策略的有效性。

### 2.2 YOLO 系列轻量化检测方法

YOLO 系列方法将目标定位和分类整合到统一网络中，具有较高的推理效率。近年来，YOLOv5、YOLOv8 和 YOLO11 等版本在网络结构、训练策略和部署工具链方面不断发展，成为目标检测工程实践中的常用框架。对于无人机航拍任务，YOLO 系列模型能够满足实时性需求，但在小目标占比较高的场景中，轻量模型容易受到下采样和特征容量限制，导致极小目标或密集目标召回不足。

为了提升轻量 YOLO 模型在小目标场景中的表现，常见思路包括增加小目标检测层、改进 Neck 特征融合结构、引入轻量化注意力模块和采用更高输入分辨率。本文的 P2 检测分支与这一思路一致，通过引入更浅层的高分辨率特征，使检测头能够处理更小尺度目标。同时，本文并不将外部 YOLOv8n 或 YOLO11s 结果作为单一模块消融依据，而是将其作为模型版本和容量变化下的参考基线进行分层讨论。

### 2.3 小目标检测与多尺度特征融合

小目标检测的核心困难在于目标像素少、纹理细节弱和定位误差敏感。经过深层网络多次下采样后，小目标在高层特征图中的响应区域可能仅对应极少数特征点，容易造成特征表达不足。多尺度特征融合通过结合浅层空间细节和深层语义信息，可以缓解目标尺度变化问题。FPN、PAN、BiFPN 等结构均体现了多尺度特征融合思想。

在 YOLO 改进研究中，增加 P2 或 x-small 检测头是一类常见小目标增强策略。P2 分支能够利用更高分辨率特征图，提升小目标的定位可见性。但额外检测分支也会增加计算开销，并可能影响推理速度。因此，P2 分支是否值得引入，需要结合具体数据集的小目标比例、精度收益和速度成本综合判断。本文通过主线消融和后续分辨率公平对照，分析 P2 分支在 VisDrone 场景中的作用边界。

### 2.4 注意力机制与位置敏感特征表达

注意力机制通过对特征通道或空间位置进行重新加权，可以增强模型对关键区域的表达能力。通道注意力能够建模不同通道的重要性，空间注意力能够突出图像中的重要区域，而坐标注意力进一步将位置信息嵌入通道注意力建模过程。CoordAttention 通过沿水平和垂直方向分别编码特征，在保持轻量化的同时引入方向感知和位置敏感信息，适合移动端或轻量化网络。

对于无人机航拍图像，小目标通常与复杂背景交织，且目标外观信息不足。引入位置敏感注意力可能有助于增强目标区域响应，降低背景干扰。但注意力模块的效果依赖网络结构、数据分布和训练设置，不能简单假设其必然带来显著提升。本文将 CoordAttention 作为 P2 结构后的轻量补充模块，并通过消融实验分析其实际贡献。当前已完成结果显示，CoordAttention 的增益相对有限，因此后续论文表述应保持谨慎。

### 2.5 输入分辨率与训练增强策略

输入分辨率对小目标检测具有直接影响。较高输入分辨率可以增加小目标在输入图像和中间特征图中的有效像素数量，有助于提升检测器对远距离小目标的感知能力。但提高分辨率通常会增加显存占用和推理时间，因此需要结合应用场景进行权衡。本文在 YOLO11n-P2-CoordAttention 基础上测试 960 输入分辨率，并通过速度和复杂度结果分析其成本。

数据增强也是影响小目标检测的重要因素。Mosaic、随机缩放、Copy-Paste 和随机擦除等增强策略能够提高样本多样性，但不合适的增强强度也可能进一步压缩或遮挡小目标。本文保留小目标友好增强作为训练策略消融，设置包括更早关闭 mosaic、减小 scale 范围、轻量 Copy-Paste 和关闭随机擦除。当前结果显示，该策略相较 baseline 有提升，但未超过 P2 和 CoordAttention 结构改进，因此在期刊稿中更适合作为训练策略边界分析，而非主要贡献。
