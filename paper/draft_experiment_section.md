# 实验设置与结果分析草稿

本文实验结果均来自项目中的真实训练日志、验证结果文件或由脚本从结果文件导出的统计表。主结果表来源为 `paper/tables/main_comparison_for_paper.csv`，消融实验来源为 `paper/tables/ablation_results.csv`，速度测试来源为 `paper/tables/speed_results.csv`，类别级指标来源为 `paper/tables/per_class_results.csv`。在未获得 VisDrone 官方评测服务器返回结果前，本文不报告官方 test-dev/test-challenge AP。

## 实验设置

实验基于 Ultralytics YOLO11n 框架，在 VisDrone2019-DET 数据集上开展无人机航拍小目标检测研究。数据配置文件为 `configs/dataset/visdrone.yaml`，训练集、验证集和测试集均采用 YOLO 格式组织。本文以原始 YOLO11n 作为基线模型，并依次评估 P2 高分辨率检测头、CoordAttention 注意力模块、输入分辨率提升以及小目标友好数据增强策略对检测性能的影响。

所有主实验均训练 100 个 epoch，并使用各实验对应输出目录中的 `results.csv` 统计最终指标与最佳指标。模型复杂度由模型结构日志和权重文件统计得到，推理速度使用 `tools/benchmark_speed.py` 在相同设置下对验证集图像进行单图推理计时。速度测试采用 10 次预热和 100 张样本，报告 wall-clock 平均延迟、FPS 以及 Ultralytics 返回的预处理、推理和后处理耗时。

## 主实验结果

与原始 YOLO11n 相比，引入 P2 高分辨率检测头后，模型参数量从 2.592M 增加到 2.894M，GFLOPs 从 6.5 增加到 10.7，但最佳 mAP50 从 0.32153 提升到 0.33013，最佳 mAP50-95 从 0.18238 提升到 0.19012。这说明更浅层的高分辨率特征对 VisDrone 中密集小目标检测具有正向作用。

在 P2 模型基础上加入 CoordAttention 后，参数量小幅增加到 2.904M，GFLOPs 保持 10.7，最佳 mAP50 和 mAP50-95 分别达到 0.33073 和 0.19044。相较 P2 模型，CoordAttention 的提升幅度较小，但在几乎不增加计算量的情况下带来稳定增益，可作为轻量级注意力增强模块。

进一步将输入分辨率提升到 960 后，YOLO11n-P2-CoordAttention-960 获得当前所有已完成实验中的最佳性能，最佳 mAP50 为 0.41996，最佳 mAP50-95 为 0.25174。相较原始 YOLO11n，mAP50 提升 0.09843，mAP50-95 提升 0.06936；相较 YOLO11n-P2，mAP50 提升 0.08983，mAP50-95 提升 0.06162。该结果表明，在 VisDrone 这类小目标占比较高的数据集上，输入分辨率是影响检测精度的重要因素。

## 消融实验分析

本文消融实验围绕“小目标检测能力增强”逐步展开。首先，P2 检测头利用更高分辨率的浅层特征增强小尺度目标定位能力，是相较基线模型的主要结构改进。其次，CoordAttention 在通道注意力中引入方向位置信息，能够在较低额外开销下进一步增强目标区域表达。最后，输入分辨率由 640 提升至 960 后，目标在特征图上的有效像素占比增加，带来了最显著的性能提升。

针对数据增强策略，本文还进行了 YOLO11n-P2-CoordAttention-SmallObjAug 实验。该实验采用 `close_mosaic: 20`、`scale: 0.35`、`copy_paste: 0.1` 和 `erasing: 0.0`，目的是减少过度缩放与擦除对小目标可见性的破坏，并在训练后期更早关闭 mosaic，使模型更充分适应真实图像分布。该模型最佳 mAP50 为 0.32780，最佳 mAP50-95 为 0.18699，相较原始 YOLO11n 分别提升 0.00627 和 0.00461，但低于 YOLO11n-P2 和 YOLO11n-P2-CoordAttention。该结果说明小目标友好增强具有一定正向作用，但在当前设置下不如结构层面的 P2 特征增强和高分辨率输入有效。

## 推理速度与复杂度分析

速度测试结果显示，原始 YOLO11n 的平均 wall-clock 延迟为 40.092 ms，FPS 为 24.94。YOLO11n-P2 的平均延迟为 43.642 ms，FPS 为 22.91；YOLO11n-P2-CoordAttention 的平均延迟为 45.768 ms，FPS 为 21.85。可以看出，P2 检测头和 CoordAttention 带来了一定推理开销，但整体仍保持可接受的单图推理速度。

YOLO11n-P2-CoordAttention-960 由于输入分辨率更高，平均 wall-clock 延迟增加到 50.811 ms，FPS 降至 19.68，但其最佳 mAP50 和 mAP50-95 均显著高于 640 输入模型。因此，该模型体现出更偏向精度的折中：在仍具备较快推理能力的前提下，明显增强了无人机小目标检测性能。
