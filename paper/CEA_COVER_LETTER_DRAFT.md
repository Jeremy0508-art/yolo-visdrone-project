尊敬的《计算机工程与应用》编辑部：

您好！

我们拟向贵刊投稿论文《【中文题名待填写】》（英文题名：【英文题名待填写】）。本文面向 VisDrone2019-DET 无人机航拍目标检测任务，针对小目标占比高、目标密集、遮挡频繁以及类别外观相似等问题，围绕轻量化 YOLO 检测模型开展改进与系统实验分析。

本文以 Ultralytics YOLO11n 为基础模型，重点评估 P2 高分辨率检测分支、CoordAttention 注意力模块、960 输入分辨率以及小目标友好数据增强策略在航拍小目标检测中的作用。实验部分在统一数据处理、训练配置和验证流程下，整理了 YOLO11n、YOLO11n-P2、YOLO11n-P2-CoordAttention、YOLO11n-P2-960、YOLO11n-P2-CoordAttention-960、YOLO11n-P2-CoordAttention-SmallObjAug，以及 YOLOv5n、YOLOv8n、YOLO11s 等参考模型的验证集结果，并补充了参数量、GFLOPs、模型大小、推理速度、尺度分组匹配、类别级结果、可视化结果和失败案例分析。

当前验证集结果表明，YOLO11n-P2-960 在 nano 级轻量模型中取得较好的精度-速度折中，best mAP50 为 0.42361，best mAP50-95 为 0.25552，单图 wall-clock 推理速度为 55.68 FPS。更大容量的 YOLO11s-960 取得更高验证集精度，说明模型容量仍是提升检测性能的重要因素。因此，本文在结论中将 YOLO11n-P2-960 表述为轻量模型中的折中方案，并将 YOLO11s-960 作为容量更大的参考上界，而不宣称本文方法全面优于所有 YOLO 模型。

本文的实验数值均来自真实训练日志、验证输出、测速结果和 `paper/tables/` 汇总文件；论文材料中同时保留了复现实验命令、数值来源审计和公开 GitHub 项目链接，便于后续审稿核查。当前结果基于 VisDrone 验证集，尚未写入官方 test-dev AP。若后续获得 VisDrone 官方评测平台返回结果，将按真实结果补充或修订。
