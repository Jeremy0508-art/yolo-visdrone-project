# P2 模型定性案例分析

本文件整理 YOLO11n-P2 在 VisDrone 验证集上的典型成功案例与失败案例。案例由脚本按验证集标签和 P2 预测结果筛选得到，指标为逐图粗略统计，用于辅助报告中的定性分析。

说明：这里的 `Matched` 和 `Recall` 使用同类别 IoU>=0.5 的简单贪心匹配计算，目的是帮助挑选案例，不替代 Ultralytics 官方验证指标。

## 报告可用分析

YOLO11n-P2 在目标尺度适中、遮挡较少的航拍场景中表现较好，能够较完整地检出车辆、行人和摩托车等目标。例如 `0000360_06861_d_0000748.jpg` 中，P2 对 25 个标注目标匹配到 24 个，说明增加 P2 高分辨率检测头后，模型对较清晰的小目标具有较好的召回能力。

在目标密集场景中，P2 可以检出大量目标，但仍存在漏检。例如 `0000295_02900_d_0000034.jpg` 包含 212 个标注目标，其中小面积目标比例较高，模型粗略召回约 0.55。这说明 P2 对小目标有效，但当目标高度密集、互相遮挡、类别外观相近时，仍会受到特征表达和后处理的限制。

失败案例主要集中在远距离极小目标、密集行人/车辆、低分辨率 motor 等场景。例如 `0000276_03201_d_0000523.jpg` 和 `0000001_04527_d_0000008.jpg` 中，大量目标面积很小，模型预测数量明显少于真实目标数量，体现出无人机航拍小目标检测仍存在漏检问题。

## 后续改进方向

- 在 P2 结构基础上继续尝试注意力机制，例如 P2+ECA 或更适合小目标的空间注意力模块。
- 提高输入尺寸，例如 `imgsz=960`，保留更多小目标细节。
- 优化数据增强策略，降低过强缩放对小目标的破坏。
- 针对 pedestrian、people、motor 等易混淆类别进行类别级误差分析。
